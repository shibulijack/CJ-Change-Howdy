A slick WP plugin that lets you change the "Howdy" text.

== Description ==

If you are looking for a way to change the **Howdy** text in the admin bar, then you are in the right place.

Change the Howdy text to *whatever you wish*, right from the admin panel!

**Works with Wordpress 3.3.1**

== Installation ==

1. Install through the WordPress admin or upload the plugin folder to your /wp-content/plugins/ directory 
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to the CJ Change Howdy menu under Settings tab

== Frequently Asked Questions ==

= Do I require PHP knowledge or any other coding skills to use this plugin? =

No. Changing Howdy text is as simple as filling out a text box.
